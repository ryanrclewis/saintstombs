# saintstombs — Region Content Files

This directory contains Markdown files for each geographic region, served as static assets at `/regions/*.md` for client-side rendering.

## File Format

Each file follows this structure:

- **Location paragraph**: A plain text paragraph naming the city or town.
- **Church list** (`-` items): Each item names a church or site.
- **Saint/object sub-list** (`  -` items): Indented items under a church, listing saints or holy objects buried or venerated there.
